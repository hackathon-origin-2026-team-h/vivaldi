## プロジェクト概要
**fumumu**（プロジェクト名: vivaldi）は、学会発表などのイベントで利用する、リアルタイム文字起こし＆意訳アプリケーションです。
「ふむふむ...（低い理解度）」から「わかった！」へ繋げることをコンセプトとし、発表者のスピーチ内容を動的に、聴講者一人ひとりの理解度に合わせて噛み砕き、理解を助けます。

## 背景・課題・解決されること
専門的な学会発表やプレゼンテーションにおいて、聴講者の知識レベルは様々です。
既存の発表形式には、以下のような課題がありました。

- **専門用語の壁**: 発表者が使う専門用語が理解できず、話についていけなくなる
- **質問のハードル**: 発表中に「わからない」と表明することが難しく、そのまま進行してしまう
- **一律の情報提供**: 全員に同じ説明がされるため、初心者には難しく、専門家には冗長になる

fumumuは、これらの課題を以下の方法で解決します。

- **リアルタイム意訳**: 発表者の音声をリアルタイムで文字起こしし、さらにLLMを用いて平易な文章に意訳します。
- **パーソナライズ**: 聴講者が「わからなかった」ボタンを押すことで、その人の理解度や特性をシステムが学習し、次回以降の意訳をその人に合わせて調整します。
- **シームレスな参加**: アプリのインストールは不要。QRコードを読み取るだけで、ブラウザからすぐに聴講を開始できます。

## プロダクト説明

fumumuは、発表者と聴講者の2つのロールで構成されるWebアプリケーションです。

### 発表者ビュー
- トークセッションを作成し、聴講者用のQRコードとURLを発行します。
- 「発表を開始する」ボタンを押すと、マイクから音声を取得し、Deepgram APIを用いてリアルタイムに文字起こしを行います。
- 文字起こしされたテキストは、Anthropic API (Claude) を用いてフィラー（えー、あの、等）の除去や文章の整形が行われ、サーバーを介して聴講者に配信されます。

### 聴講者ビュー
- 発表者が提示したQRコードを読み取ることで、専用のセッションページ（`/[sessionId]`）にアクセスします。
- 発表者の発言内容と、それを意訳した文章がリアルタイムに表示されます。
- 各文章に対して「理解できた（電球アイコン）」「わからなかった（？アイコン）」のリアクションボタンがあり、フィードバックを送ることができます。
- 「わからなかった」というフィードバックはローカルストレージ（Persona）に蓄積され、以降の意訳処理において、そのユーザーの知識レベルに合わせたパーソナライズが行われます。

## 注力したポイント

### アイデア面
最大のこだわりは、**「聴講者の理解度に応じたパーソナライズ」**です。
単なる文字起こしツールではなく、ユーザーのリアクション（わからなかった）をコンテキストとして蓄積し、LLMのプロンプトに反映させることで、「その人にとってわかりやすい」意訳を動的に生成する仕組みを実現しました。

### デザイン面
「AI感」や「無機質なテック感」を排除し、**「親しみやすさ」と「やさしさ」**を重視しました。
- 背景色にはやわらかいクリーム色（`#FFF4E5`）、アクセントカラーには温かみのあるオレンジ（`#E98527`）を採用。
- フォントには丸ゴシック（Kosugi Maru）を使用し、圧迫感のないUIを構築。
- 聴講者ビューでは、最新の発言に自動でスクロール追従する機能や、リアクションボタンの直感的なアニメーションなど、心地よいユーザー体験（UX）にこだわりました。

### 技術面
- **リアルタイム通信**: Cloudflare Workers と Durable Objects を活用し、Server-Sent Events (SSE) による低遅延なリアルタイム配信を実現しています。
- **音声処理**: DeepgramのWebSocket APIを使用し、ブラウザのAudioContextから直接音声ストリームを送信することで、高速な文字起こしを行っています。
- **エッジコンピューティング**: OpenNextを用いてNext.jsアプリケーションをCloudflare Workers上にデプロイし、グローバルで高速なレスポンスを提供します。

## 使用技術

### フロントエンド
- Next.js (App Router)
- React
- Tailwind CSS
- GSAP (アニメーション)

### バックエンド / インフラ
- Cloudflare Workers
- Cloudflare Durable Objects (セッション管理・リアルタイム通信)
- OpenNext (Next.js on Cloudflare)

### 外部API
- Deepgram API (リアルタイム音声認識・文字起こし)
- Anthropic API / Claude (文章の整形・パーソナライズ意訳)

## 開発環境のセットアップ

```bash
# 依存関係のインストール
pnpm install

# 環境変数の設定
cp .env.example .env
# .env ファイルに DEEPGRAM_API_KEY と ANTHROPIC_API_KEY を設定してください

# 開発サーバーの起動
pnpm dev
```
